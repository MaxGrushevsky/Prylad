# Генератор QR кодов

Современный веб-сайт для генерации QR кодов с возможностью кастомизации. Все вычисления выполняются на фронтенде.

## Особенности

- ✅ Генерация QR кодов на фронтенде (без сервера)
- ✅ Полностью бесплатно - все функции доступны
- ✅ Кастомизация цветов QR кода и фона
- ✅ Настройка размера (200px - 600px)
- ✅ Выбор уровня коррекции ошибок
- ✅ SEO оптимизация
- ✅ Подготовлено место для рекламных блоков
- ✅ Адаптивный дизайн
- ✅ Современный UI/UX

## Технологии

- **Next.js 14** - React фреймворк с SSR для SEO
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **qrcode** - Библиотека для генерации QR кодов на фронтенде

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```bash
npm run build
npm start
```

## Структура проекта

```
├── app/
│   ├── layout.tsx      # Главный layout с SEO мета-тегами
│   ├── page.tsx        # Главная страница
│   └── globals.css     # Глобальные стили
├── components/
│   ├── QRCodeGenerator.tsx  # Компонент генератора QR кодов
│   └── AdBanner.tsx         # Компонент рекламного блока
└── ...
```

## SEO оптимизация

- Мета-теги для поисковых систем
- Open Graph теги для социальных сетей
- Структурированные данные (JSON-LD)
- Семантическая HTML разметка
- SSR для лучшей индексации
